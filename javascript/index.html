Javascript practice - exercises from Hackerrank

surface area

Exercise: calculating the surface of an object consisting of square blocks of differing heights. Sample 2d array as height map:

[[1, 3, 4], [2, 2, 3], [1, 2, 4]]

quicksort

Implementing quicksort. Partially tested with this exercise. Sample array:

[0, -3, 6, 4, -10, 8, -5, 2, -7]

icecream parlor

Exercise: finding, if exists, 2 numbers in an array with a fixed sum, represented with their 1-based index. Sample input:

[1, 4, 5, 3, 2], to sum to 5

Jim and the orders

Exercise: calculating the order a group of restaurant customers will be served, based on the calculated time preparing their order will take. Sample input:

  • customer id: (always like) [1, 2, 3, 4, 5]
  • "order #": (time taken by preparation step 1) [8, 5, 6, 2, 4]
  • "prep time": (time taken by preparation step 2) [3, 6, 2, 3, 3]

organizing containers

Exercise: given a row of containers, full of balls of different colors, determine if one can reach a state where each container has only one color, by swapping pairs of balls. Sample input as a 2d array (containers x quantity of balls of each color in them):

[[0, 2, 1], [1, 1, 1], [2, 0, 0]]

cavity map

a solution to Exercise: given a 2d array of heights, mark with X the cells which are not on the margins and higher than all their neighbours. Sample input:

["1112","1912","1892", "1234"]

almost sorted

Exercise: realizing if the input array can be transformed to sorted by reversing a subarray or swapping 2 elements. Sample array:

[1, 5, 4, 3, 2, 6]

Sherlock and array

Exercise: realizing if the input array has an element that sits between two subarrays with equal sums (empty subarrays sum to zero). Sample array:

[1, 2, 4, 0, 3]

cloudy day

Exercise: in a row of cities with varying population sizes, under sometimes-overlapping clouds of varying sizes, find the maximum number of people who could see sunlight after eliminating 1 cloud. The real challenge is constructing the objects used from the format the input is given. Sample input:

  • city populations: [10, 100]
  • city locations: [5, 100]
  • cloud locations: [4]
  • cloud ranges (radiuses): [1]

gemstones

Exercise: finding the number of letters appearing in all strings. Sample input:

["abcde", "bxa", "adbkctde", "adbkctdevvvv"]

mail delivery

I solved the problem described in this chapter, trying to use pure functions. In a village, a mail delivery robot is notified each day about any residents intending to send parcels to any other residents, and the problem to solve is planning a reasonably short trip across the village to pick up and deliver all parcels. The mail delivery robot always starts its trip from the Post Office. The input is randomly generated every time the page is refreshed.
