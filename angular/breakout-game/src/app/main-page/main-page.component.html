The game's basic idea originates with the
Breakout arcade game in 1976 (see on wikipedia).

"Time step" used in this sense, to describe the game loop.
In each version, refresh page for a different (randomly-generated) level.

  • Simpler version with a fixed time step

  • More complex version with a variable time step

    Incorporating Timm Preetz's Gist for finding if
    two line segments intersect (for collision detection).
